# filament-ai-skills

A complete Claude Code skills suite for **Filament v5 + Laravel** development.

## Install

```bash
/install github:felipeArnold/filament-ai-skills
```

## Skills Included

| Skill | Trigger |
|-------|---------|
| `filament-resource-development` | create resource, filament CRUD, new resource |
| `filament-action-development` | create action, modal, bulk action, confirmation |
| `filament-infolist-development` | infolist, view page, read-only details |
| `filament-export-development` | export Excel, PDF list, download data |
| `filament-test-resource` | test resource, Filament test suite, generate tests |
| `filament-query-audit` | audit queries filament, N+1 filament, eager load audit |
| `pre-commit-review` | before commit, pre-commit, review staged changes |
| `null-safe-check` | null safe, ?->, null check, ErrorException on null |
| `eloquent-query-check` | Model::find, Model::create, static model call |
| `pest-testing` | pest test, write test, browser test, TDD |
| `laravel-best-practices` | Laravel patterns, best practices, code review |
| `error-detective` | error, exception, bug, 500, stack trace, crash |
| `spec-creator` | create spec, plan feature, write requirements, SPEC.md |
| `query-performance-audit` | slow query, N+1, timeout, EXPLAIN, 504 |
| `deep-code-review` | deep review, security audit, code quality, SOLID |
| `livewire-development` | livewire, wire:model, wire:click, reactive component |
| `pennant-development` | feature flag, Pennant, toggle, A/B test |
| `tdd-implementation` | TDD, red-green-refactor, test-driven, implement with tests |

## Requirements

- PHP 8.2+
- Laravel 11+
- Filament v5
- Livewire v4
- Pest v4

## Stack

Built for the **Filament v5 + Livewire 4 + Tailwind 4** ecosystem.  
Skills follow PHP 8.4 patterns and Laravel 12 conventions.
